# 敌人系统文档

## 概述

敌人系统管理游戏中所有敌方单位（敌人）的属性、AI 行为、生成和掉落。敌人是战斗中的对抗方，每个敌人占据一个地图格子，拥有独立的属性、技能和 AI 逻辑。

---

## 基本属性

### 基础属性（Base Stats）

| 属性 | 说明 |
|------|------|
| 生命值（HP） | 敌人生命上限，归零时死亡 |
| 攻击力（ATK） | 影响造成的伤害值 |
| 防御力（DEF） | 减少受到的伤害 |
| 速度（SPD） | 决定行动顺序或攻击频率 |
| 射程（RNG） | 攻击范围（格数） |
| 移动力（MOV） | 每回合可移动的格数 |

### 派生属性（Derived Stats）

| 属性 | 说明 |
|------|------|
| 暴击率 | 攻击产生暴击的概率 |
| 闪避率 | 闪避攻击的概率 |
| 生命恢复 | 每回合恢复的生命值 |

---

## 敌人类型

| 类型 | 特点 |
|------|------|
| 普通 | 基础敌人，属性均衡，无特殊能力 |
| 精英 | 属性高于普通敌人，拥有 1-2 个技能 |
| Boss | 极高属性，拥有多个技能和阶段切换机制 |
| 召唤物 | 由其他敌人或技能召唤产生，属性较低，存在时间有限 |

---

## AI 行为

### 行为模式

| 模式 | 说明 |
|------|------|
| 直线冲锋 | 沿固定路径向要塞前进，无视沿途目标 |
| 索敌攻击 | 在攻击范围内优先攻击最近的敌方单位 |
| 集火 | 优先攻击血量最低的目标 |
| 游走 | 随机移动，遇到目标后切换为索敌攻击 |
| 守卫 | 固定不动，攻击进入射程的目标 |

### 决策流程

1. 检测攻击范围内是否存在目标
2. 有目标 → 判断技能是否可用 → 施放技能 / 普通攻击
3. 无目标 → 判断是否可移动 → 向目标方向移动
4. 无法移动或攻击 → 待命

## 生成与波次

### 波次系统

- 敌人以**波次**（Wave）形式生成。
- 每波包含一个预定义的敌人编组，含多种类型和数量。
- 波次之间的间隔时间可配置。
- 全部波次消灭后判定胜利。

### 生成位置

- 敌人从地图边缘的指定**入口点**生成。
- 可设置多个入口点，每波选择其中一个或随机分布。
- 生成后自动进入 AI 行为循环。

### 难度缩放

- 敌人属性根据关卡进度和玩家实力动态调整。
- 缩放因子：`baseStats × (1 + waveNumber × scaleFactor)`。

## 敌人状态

### 基础状态

- **生成（Spawning）**：进入战场，播放入场动画。
- **移动（Moving）**：按照 AI 行为路径移动。
- **攻击（Attacking）**：进行攻击动画和伤害判定。
- **技能施放（Casting）**：施放技能。
- **受击（Hit）**：受到攻击时的硬直状态。
- **死亡（Dead）**：生命值为 0，播放死亡动画后移除。

### 增益 / 减益状态（Buff / Debuff）

复用角色系统定义的 Buff/Debuff 体系（眩晕、减速、灼烧等）。

---

## 掉落系统

- 敌人被击败时掉落资源（金币、经验、装备等）。
- 掉落概率和数量由敌人类型和等级决定。
- Boss 和精英敌人保证掉落稀有物品。

| 敌人类型 | 掉落 |
|---------|------|
| 普通 | 基础资源，低概率掉落 |
| 精英 | 中等资源，中概率掉落装备/技能书 |
| Boss | 大量资源，必定掉落稀有物品 |

---

## 与现有系统的关系

### 与地图系统的交互

- 敌人通过格子坐标 `(x, y)` 定位在地图上。
- 移动和寻路依赖地图格子和地形数据。
- 不可通行地形阻挡敌人移动。

### 与战斗系统的交互

- 敌人参与战斗系统的伤害计算和技能判定。
- 敌人作为攻击方时，伤害计算逻辑与角色一致。
- 敌人作为受击方时，适用相同的防御和减伤规则。

### 与渲染系统的交互

- 敌人使用 `VisualDisplayLayer::Character` 渲染层级（z = 1.0），与角色同一层。

## 数据结构示意

```
Enemy {
  id: String,
  name: String,
  enemyType: EnemyType,        // Normal | Elite | Boss | Summon
  level: u32,
  baseStats: {
    hp: u32,
    atk: u32,
    def: u32,
    spd: u32,
    rng: u32,
    mov: u32,
  },
  skills: [Skill],
  buffs: [Buff],
  aiMode: AiMode,              // Charge | Seek | FocusFire | Roam | Guard
  gridPosition: { x, y },
  dropTable: DropTable,
}
```
