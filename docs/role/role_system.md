# 角色系统文档

## 概述

角色系统管理游戏中所有可操控单位（角色）的属性、状态和行为。角色是战斗中的基本单元，每个角色占用一个地图格子，拥有独立的属性、装备和技能。

---

## 基本属性

### 基础属性（Base Stats）

| 属性 | 说明 |
|------|------|
| 生命值（HP） | 角色生命上限，归零时死亡 |
| 攻击力（ATK） | 影响造成的伤害值 |
| 防御力（DEF） | 减少受到的伤害 |
| 速度（SPD） | 决定行动顺序或移动范围 |
| 射程（RNG） | 攻击范围（格数） |
| 移动力（MOV） | 每回合可移动的格数 |

### 派生属性（Derived Stats）

| 属性 | 说明 |
|------|------|
| 暴击率 | 攻击产生暴击的概率 |
| 闪避率 | 闪避敌方攻击的概率 |
| 生命恢复 | 每回合 / 每秒恢复的生命值 |

---

## 角色类型

| 类型 | 特点 |
|------|------|
| 近战 | 高生命、高攻击、低射程（1 格） |
| 远程 | 低生命、中高攻击、远射程（2+ 格） |
| 法师 | 低生命、中攻击、范围伤害 / 法术效果 |
| 辅助 | 低攻击、治疗或增益 / 减益技能 |
| 坦克 | 极高生命、高防御、低攻击 |

---

## 角色状态

### 基础状态

- **待命（Idle）**：未行动，可执行指令。
- **移动（Moving）**：正在进行移动动画。
- **攻击（Attacking）**：正在进行攻击动画。
- **技能施放（Casting）**：正在施放技能。
- **死亡（Dead）**：生命值为 0，从场上移除。

### 增益 / 减益状态（Buff / Debuff）

| 状态 | 效果 |
|------|------|
| 眩晕 | 无法行动，持续 N 回合 |
| 减速 | 移动力减半 |
| 增伤 | 受到的伤害增加 X% |
| 护盾 | 抵挡 N 点伤害 |
| 隐身 | 无法被敌方选中 |
| 灼烧 | 每回合受到 N 点伤害 |

---

## 角色养成

### 升级

- 角色通过战斗获得经验值，经验值累积满后升级。
- 每升一级提升基础属性（HP、ATK、DEF 等），并可解锁新技能或强化已有技能。

### 升星

- 收集多个相同角色卡牌可升星。
- 升星大幅提升属性并解锁被动技能槽位。

### 装备

- 每个角色可装备武器、防具、饰品各一件。
- 装备提供额外属性加成或特殊效果。

---

## 技能系统

### 技能类型

| 类型 | 说明 |
|------|------|
| 主动技能 | 战斗中有 CD 可主动施放的技能 |
| 被动技能 | 战斗中持续生效或条件触发的技能 |
| 普攻 | 基础攻击行为，部分角色普攻附带特效 |

### 技能效果

- **伤害类**：单体 / 范围 / 弹射 / 穿透
- **控制类**：眩晕 / 嘲讽 / 沉默 / 束缚
- **辅助类**：治疗 / 护盾 / 增益 / 减益
- **位移类**：冲刺 / 击退 / 交换位置

---

## 站位规则

- 每个角色占据一个地图格子（64×64 px）。
- 一个格子最多只能站一个角色。
- 角色可放置于**场上**（地图格子）或**备战席**（地图下方 8 格）。
- 出战角色总数量受等级 / 关卡限制。

---

## 数据结构示意

```
Character {
  id: String,
  name: String,
  level: u32,
  star: u32,
  exp: u32,
  baseStats: {
    hp: u32,
    atk: u32,
    def: u32,
    spd: u32,
    rng: u32,
    mov: u32,
  },
  equipment: [Weapon, Armor, Accessory],
  skills: [Skill],
  buffs: [Buff],
  gridPosition: { x, y } | null,
  isOnBench: bool,
}
```

---

## 与地图系统的交互

- 角色通过格子坐标 `(x, y)` 定位在地图上。
- 移动、技能范围计算依赖地图格子坐标和地形数据。
- 不可通行地形（墙壁、水域等）影响角色移动与站位。
- 渲染顺序：地图层 → 角色层 → 特效层。
